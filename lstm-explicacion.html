import torch.nn as nn

class ModeloGRU(nn.Module):
    def __init__(self, input_dim, hidden_dim, 
                layer_dim, output_dim, dropout_prob):
        super(ModeloGRU, self).__init__()
        
        # Dimensiones
        self.hidden_dim = hidden_dim
        self.layer_dim = layer_dim
        
        # Capas GRU
        self.gru = nn.GRU(
            input_dim, hidden_dim, 
            layer_dim, batch_first=True,
            dropout=dropout_prob
        )
        
        # Capa de salida
        self.fc = nn.Linear(hidden_dim, output_dim)
    
    def forward(self, x):
        # Inicializar estado oculto con ceros
        h0 = torch.zeros(self.layer_dim, 
                        x.size(0),

        # √çndice de la √∫ltima secuencia temporal
        out = out[:, -1, :]
        
        # Capa de salida
        out = self.fc(out)
        return out

Las arquitecturas encoder-decoder (codificador-decodificador) son un tipo especial de redes neuronales dise√±adas para transformar una secuencia de entrada en otra secuencia de salida, potencialmente de diferente longitud.

¬øC√≥mo funcionan?

La arquitectura consta de dos componentes principales:

Esta estructura es ideal para tareas donde tanto la entrada como la salida son secuencias, como traducci√≥n autom√°tica, resumen de texto, o conversi√≥n de voz a texto.
